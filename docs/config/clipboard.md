# Clipboard

_Category `clipboard:`_

| Property        | Description                          | Type      | Default |
| --------------- | ------------------------------------ | --------- | ------- |
| `enabled`       | enable of clipboard                  | `boolean` | `true`  |
| `max-items`     | number of maximum items in clipboard | `integer` | `50`    |
| `image-preview` | enable of image preview              | `boolean` | `true`  |
| `width`         | width of launcher window             | `integer` | `400`   |
| `height`        | height of launcher window            | `integer` | `500`   |
