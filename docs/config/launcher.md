# Launcher

_Category `launcher:`_

| Property  | Description                  | Type      | Default |
| --------- | ---------------------------- | --------- | ------- |
| `width`   | width of launcher window     | `integer` | `400`   |
| `height`  | height of launcher window    | `integer` | `500`   |
| `columns` | count of columns in launcher | `integer` | `1`     |
