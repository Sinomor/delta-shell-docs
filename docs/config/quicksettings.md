# Quick Settings

_Category `quicksettings:`_

| Property  | Description                            | Type       | Default                                                     |
| --------- | -------------------------------------- | ---------- | ----------------------------------------------------------- |
| `buttons` | the list of buttons that display at qs | `string[]` | `["network", "bluetooth", "notifications", "screenrecord"]` |
| `sliders` | the list of sliders that display at qs | `string[]` | `["volume", "brightness"]`                                  |

## Buttons

Available buttons: `network`, `bluetooth`, `notifications`, `weather`, `screenrecord`, `volume`, `microphone`

## Sliders

Available sliders: `volume`, `brightness`, `mircophone`
